# Football API 
A simple RESTful API built with **Express.js** and **MongoDB** (using **Mongoose**) for retrieving football (soccer) data â€” matches, teams, players, etc.

## Features

- CRUD operations for teams, players, matches  
- JSON-based API  
- Organized route structure  
- Data models (e.g. using `models/`)  
- Easily extensible

## Get Started

**1. Clone this repository**  
 ```bash
 git clone https://github.com/Kally-Dossa/football-api.git
 cd football-api
```

**2. Install dependencies**
 ```bash
npm install
```

**3. Setup configuration (e.g. database URI, port)**

**4. Run**
 ```bash
node app.js
```
